# ArknightsName
《明日方舟》名称

## Tip

 - NPC名称通过剧情文本提取，可能有误
 - Pool id各服务器均不相同
 - Memory名称可能重复(同set)

## 文件说明

| 文件夹 | 说明 |
| --- | --- |
| [data](./data) | json数据 |
| [version](./version) | 对应的版本号，可用于判断是否更新 |
| [table](./table) | 数据可视化
